# Climate Change Impact on Agriculture (1990-2024)

## Description  
This project investigates how changes in climate variables (temperature, rainfall) over the period 1990 to 2024 have affected agricultural productivity across different regions. The results are presented via an interactive Power BI dashboard.

## Key Features  
- Data cleaning & preprocessing of climate + agriculture datasets  
- Interactive Power BI dashboard with filters (year, region, crop)  
- Visual insights: trend analysis, correlation heatmaps, regional impact maps  
- Exportable summary of findings  

## Tools & Technologies  
- Power BI Desktop  
- Excel (Power Query)  
- Datasets: `climate_exchange.xlsx` (climate data), agriculture yield data  
- Version control: Git & GitHub  

## Dataset  
The climate data spans 1990-2024 covering variables like average temperature, rainfall anomalies. The agriculture dataset covers major crops and yield changes over the same period.

## How to Use  
1. Download `Project_Rahul02.pbix`.  
2. Open in Power BI Desktop.  
3. Use filters to explore by region, year range, crop type.  
4. View visuals: Trend chart → Map view → Correlation matrix.  

## Key Findings  
- Region X: Avg. temp increased by Y°C which correlated with a Z% drop in Crop A yield.  
- Rainfall deviation has a strong negative relationship (r=–0.65) with Crop B’s yield in drought-prone zones.
